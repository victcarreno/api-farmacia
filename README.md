# 🏥 API de Gestión de Inventario Farmacéutico

API RESTful desarrollada en **Java Spring Boot** para el control de stock de medicamentos. Este proyecto combina conocimientos técnicos de backend con experiencia en el dominio farmacéutico.

## 🚀 Tecnologías Utilizadas

- **Java 17**
- **Spring Boot 3.x**
- **Spring Data JPA**
- **H2 Database** (desarrollo)
- **Maven**
- **Git/GitHub**

## 📋 Endpoints Disponibles

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | `/api/productos` | Listar todos los productos |
| GET | `/api/productos/{id}` | Obtener producto por ID |
| POST | `/api/productos` | Crear nuevo producto |
| PUT | `/api/productos/{id}` | Actualizar producto |
| DELETE | `/api/productos/{id}` | Eliminar producto |
